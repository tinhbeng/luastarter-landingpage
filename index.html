Exceptional features

Multi-chain offering

LuaStarter supports both Ethereum and TomoChain pools, providing flexible options for investors to partialpate in both chains at the same time.

Tier Guaranteed System Model

TGS model is based on tiers’ total counts, enabling investors not only to be guaranteed in the IDO but also utilize all of their tokens. The flexibility of model also provides opportunities for all investors to buy tokens in fair and efficient manner.

Ecosystem Integration

LuaStarter allows both project creators and investors to utilize all of existing LuaSwap and TomoChain ecosystem, including briding, swapping, farming, on-chain orderbook, portfolio management.

Tier Guaranteed System

Tier Guaranteed System guarantees the number of tokens to be allocated to each tier’s member for every IDO pools.
